# Number Guessing Game (Python)

## Description
A simple Python mini project where the user guesses a randomly generated number between 1 and 10.
This is a beginner-level Python mini project where the program randomly generates a number between 1 and 10, and the user tries to guess it. The game continues until the correct number is guessed. This project helps in understanding Python basics such as loops, conditional statements, user input, and the random module.

## Technologies Used
- Python

## Features
- Random number generation
- User input
- Loop and conditional statements

## How to Run
1. Install Python
2. Run:
   python guess.py

## Output
User guesses until the correct number is entered.
